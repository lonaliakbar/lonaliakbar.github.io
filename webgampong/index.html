Tanya Jawab

Pertanyaan Umum (FAQ)

Apakah layanan GampongWeb benar-benar gratis?
+
Ya, 100% gratis selamanya untuk gampong yang berada di wilayah Aceh. Kami menanggung biaya server dan pemeliharaan sistem.
Apa syarat mendaftar website gampong?
+
Pendaftar harus merupakan aparatur resmi gampong (Geuchik, Sekdes, atau operator desa yang ditunjuk) dengan melampirkan SK pengangkatan untuk verifikasi admin.
Apakah bisa menggunakan domain .desa.id sendiri?
+
Saat ini sistem kami menggunakan subdomain .gampong.id secara default agar mempercepat proses pengaktifan. Fitur *custom domain* .desa.id sedang dalam tahap pengembangan.
Bagaimana jika saya kesulitan mengelola websitenya?
+
Kami menyediakan panduan lengkap berupa modul PDF dan video tutorial. Selain itu, ada grup Telegram khusus admin gampong untuk saling berdiskusi dan bantuan teknis.
